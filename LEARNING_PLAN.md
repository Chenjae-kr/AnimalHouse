# 3D 동물 키우기 웹사이트 학습 계획서

## 📚 학습 목표
Vue.js와 Three.js를 활용하여 HammyHome과 유사한 3D 인터랙티브 동물 키우기 웹사이트를 개발하면서 프론트엔드 개발 역량을 체계적으로 습득합니다.

### 막힐 때 대처법
1. **에러 메시지 꼼꼼히 읽기**: 대부분의 해답은 에러 메시지에 있습니다
2. **작은 단위로 테스트**: 복잡한 기능은 작은 부분으로 나눠서 테스트
3. **콘솔 활용**: `console.log()`로 변수 상태 확인
4. **Vue DevTools 사용**: 컴포넌트 상태를 시각적으로 확인
5. **Three.js Inspector**: 3D 씬 디버깅에 활용

### 효율적인 학습을 위한 조언
- **완벽주의 버리기**: 처음부터 완벽한 코드를 쓸 필요 없습니다
- **반복이 중요**: 같은 개념을 여러 번 연습하세요
- **실제로 만들기**: 튜토리얼만 보지 말고 직접 만들어보세요
- **커뮤니티 참여**: 질문하고 답변하며 성장하세요
- **꾸준함**: 하루 1시간이라도 매일 하는 것이 중요합니다
